#### **RFC-006: 本地音频文件解码与播放**

**功能定位**
- 定义音频文件格式支持清单（MP3/FLAC/WAV/OGG）
- 规范 FileAudioProvider 的实现规范
- 明确元数据提取与缓存策略

**核心要求**
- 支持主流格式：MP3、FLAC、WAV、OGG Vorbis、AAC
- 使用 NAudio / TagLibSharp 进行解码和元数据提取
- 实现进度条拖拽、快进快退功能
- 元数据缓存策略（避免重复解析）
- 畸形文件容错处理（损坏的MP3头）

**设计范围**
- 支持的音频格式详细列表
- FileAudioProvider 实现规范
- 元数据提取与缓存机制
- 播放进度管理（CurrentTime、Duration）
- 音频格式检测与自动选择合适解码器
- 播放列表加载性能优化

**交付物**
- 音频格式支持矩阵
- FileAudioProvider 设计文档
- 元数据缓存实现示例
- 格式检测与解码器选择算法
- 错误处理与降级方案
