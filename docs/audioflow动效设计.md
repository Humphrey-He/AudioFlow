# AudioFlow 灵动动效设计

## 1. 动效设计理念

### 1.1 核心原则
- **响应性**: 动效必须跟随音频数据实时响应，不可延迟
- **流畅性**: 目标 60fps，GPU 加速优先
- **克制**: 动效服务于可视化，不可喧宾夺主
- **性能**: 移动端也要流畅运行

### 1.2 性能预算
- 每帧渲染时间: < 8ms (留 8ms 给其他任务)
- Canvas 操作: 最小化 `fillRect`/`stroke` 调用次数
- 对象分配: 避免每帧 new 对象，使用对象池

## 2. 已实现动效

### 2.1 Glow (发光效果)
```
触发条件: magnitude > -20dB
效果:
  - 在条形顶部添加高光
  - 发光强度 = (magnitude + 60) / 40
  - 颜色: 白色到青色渐变
  - shadowBlur 动态调整
```

### 2.2 Reflection (倒影)
```
位置: 主条形下方 10px 间隔
高度: 主条形的 30%
透明度: 0.15
实现: LinearGradient 从颜色到透明
```

### 2.3 Peak Hold (峰值保持)
```
行为:
  - 显示当前帧最高能量点
  - 白色圆点标记峰值位置
  - 保持 1.5 秒 (hold)
  - 然后 0.5 秒衰减到当前值 (decay)
视觉效果:
  - 白色圆点带光晕
```

### 2.4 Background Pulse (背景脉动)
```
检测: 低频段 (20Hz-200Hz) 平均能量
效果:
  - 背景色随低频能量脉动
  - 脉动周期: 跟随低频节奏 (正弦波)
  - 透明度变化: 0 ~ 0.15
```

### 2.5 Center Line (中心线)
```
行为: 连接所有条形中点的平滑曲线
效果:
  - 线条粗细: 2px
  - 颜色: 半透明紫色
  - 带阴影发光
```

## 3. 扩展动效计划

### 3.1 Peak Trail (峰值轨迹)
```
行为: 峰值点随时间移动形成轨迹
实现:
  - 保留最近 60 帧的峰值位置
  - 用半透明线连接
  - 形成"山脊"效果
状态: 待实现
```

### 3.2 Radial Spectrum (环形频谱)
```
布局:
  - 条形从中心向四周展开
  - 形成圆形/半圆形频谱
  - 低频在中心，高频在外侧
状态: 待实现
```

### 3.3 Energy Ring (能量环)
```
行为: 圆形环随低频能量扩张
实现:
  - 环半径 = 基础半径 + 低频能量 * 系数
  - 环宽度: 3px
  - 颜色: 半透明渐变
状态: 待实现
```

### 3.4 Particles (粒子效果)
```
触发: magnitude 突然增加超过阈值
行为:
  - 从峰值点发射粒子
  - 粒子向上/两侧飞散
  - 生命周期: 0.5 秒
颜色: 与当前条形颜色一致
状态: 待实现
```

## 4. 预设方案

| 预设 | Glow | Reflection | Peak | Pulse | Line | 适用场景 |
|------|------|------------|------|-------|------|---------|
| Default | ✓ | ✓ | ✓ | - | - | 日常使用 |
| Bassic | ✓ | ✓ | ✓ | ✓ | ✓ | 电子音乐 |
| Vivid | ✓ | ✓ | ✓ | ✓ | ✓ | 展示/截图 |
| Minimal | - | - | - | - | - | 省电模式 |

## 5. 性能优化

### 5.1 渲染优化规则
- 预创建 Gradient 对象，尺寸变化时重建
- 使用 Path2D 批量绘制
- 非关键动效降低更新频率 (每2帧更新)

### 5.2 状态管理
- 动效状态独立于数据状态
- 使用 requestAnimationFrame 节流
- Canvas 尺寸变化时重建所有预计算对象

## 6. UI 控制

```
┌─ Effects ──────────────────────────────────┐
│ [Glow] [Reflection] [Peak] [Pulse] [Line] │
│ Preset: [Default ▼]                         │
└────────────────────────────────────────────┘
```

每个 effect 有对应的 tooltip 说明:
- Glow: 为高能量条形添加发光效果
- Reflection: 显示条形倒影增加层次感
- Peak: 保持峰值位置便于观察
- Pulse: 背景随低频能量脉动
- Line: 绘制条形中点连线
