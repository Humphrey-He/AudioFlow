## 简介

AudioFlow 是一款实时音频频谱分析器和可视化工具。它能够：
- 实时捕获系统音频、麦克风输入或本地音频文件
- 执行 FFT 频谱分析
- 以可视化方式展示音频频率分布
- 支持多种可视化模式（频谱、瀑布图、3D、极坐标等）
- 支持多人派对共享观看
- 内置节奏游戏娱乐功能

### 系统要求

| 要求 | 最低配置 | 推荐配置 |
|------|---------|---------|
| 操作系统 | Windows 10+ | Windows 11 |
| 内存 | 4GB | 8GB+ |
| 音频 | 支持系统音频捕获 | 专业声卡 |
| 网络 | 仅前端不需要 | 后端需要 localhost |

## 快速开始

### 启动方式

#### 方式一：使用启动脚本（推荐）

**Windows:**
```bash
scripts\start.bat           # 启动前后端
scripts\start.bat frontend   # 仅前端 (http://localhost:3000)
scripts\start.bat backend    # 仅后端 (http://localhost:5000)
```

**Linux/macOS:**
```bash
./scripts/start.sh           # 启动前后端
./scripts/start.sh frontend  # 仅前端
./scripts/start.sh backend   # 仅后端
```

#### 方式二：手动启动

**步骤 1：启动后端服务**

```bash
cd src/AudioFlow.Server
dotnet run
```

后端服务将在 `http://localhost:5000` 启动。

**步骤 2：启动前端（可选，用于开发）**

```bash
cd src/AudioFlow.Client
npm install
npm run dev
```

前端开发服务器将在 `http://localhost:3000` 启动。

**步骤 3：打开浏览器**

- **完整功能**：访问 `http://localhost:5000`（后端已包含前端）
- **仅前端开发**：访问 `http://localhost:3000`

**步骤 4：授予音频权限**

浏览器会请求麦克风或系统音频访问权限，请选择允许。

---

| 端口 | 服务 | 说明 |

| 3000 | Vite 前端开发服务器 | 仅前端开发时使用 |

## 功能详解

### 1. 可视化模式

AudioFlow 提供 6 种可视化模式：

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| 频谱 (Spectrum) | 经典条形频谱图 | 日常使用 |
| 瀑布图 (Waterfall) | 频谱随时间滚动的历史轨迹 | 分析声音变化趋势 |
| 波形 (Waveform) | 时域波形显示 | 观察波形形状 |
| 环形 (Polar) | 极坐标方式显示频谱 | 独特视觉效果 |
| 3D | 三维旋转频谱柱 | 展示/截图 |
| 对比 (Comparison) | 当前与平均值对比/差异曲线 | 音频分析 |

#### 瀑布图控制
- **Frames**: 历史帧数 (30-150)
- **Decay**: 衰减系数 (0.80-0.99)
- **Color**: 配色方案 (火焰/极光/科技/海洋)

#### 环形控制
- **Type**: 全圆/半圆
- **Direction**: 顺时针/逆时针

#### 3D 控制
- **Rotation Speed**: 旋转速度
- **Color Scheme**: 配色方案

### 2. 音频源切换

| 源 | 说明 |
|----|------|
| 系统 (System) | 通过后端捕获系统音频 |
| 麦克风 (Microphone) | 实时麦克风输入 |
| 文件 (File) | 本地音频文件播放 |

#### 文件播放
- 支持格式：MP3, WAV, OGG, FLAC, AAC, M4A
- 点击上传按钮选择文件
- 支持播放/暂停/停止/进度拖动/音量调节

### 3. 派对房间 (Party Room)

多人共享可视化观看功能。

**创建房间：**
1. 点击菜单中的"派对"按钮
2. 点击"创建房间"
3. 获取 6 位房间码
4. 将房间码分享给朋友

**加入房间：**
1. 点击菜单中的"派对"按钮
2. 输入房间码
3. 点击"加入"

**功能特点：**
- 房主音频实时同步给最多 10 位观众
- 音频同步延迟 < 500ms
- 观众可以看到与房主完全同步的可视化

### 4. 节拍游戏 (Beat Game)

音频驱动的节奏游戏。

**游戏规则：**
1. 点击"开始游戏"进入游戏模式
2. 观察频谱条形，找到目标指示器位置
3. 在条形达到峰值时点击
4. 根据时机准确度获得评分

**评分系统：**
| 评级 | 条件 | 分数 |
|------|------|------|
| Perfect | 峰值上升时点击 | 100分 |
| Great | 幅度 ≥ 85% | 50分 |
| Good | 幅度 ≥ 75% | 20分 |
| Miss | 未满足上述条件 | 0分 |

**连击系统：**
- 连续正确点击获得连击加成
- 连击分数 = 基础分数 + 连击数 × 5
- 最高分自动保存

### 5. 其他工具

#### 截图功能
- 点击相机图标可截取当前可视化
- 支持复制到剪贴板或下载 PNG

#### 录音功能
- 点击录制按钮可录制 WebM 视频
- 支持下载录制的视频

#### 预设分享
- 导出/导入效果预设
- 支持通过 URL 分享预设

### 6. PWA 支持

AudioFlow 支持安装为桌面应用：

1. 在浏览器中打开 `http://localhost:5000`
2. Chrome: 点击地址栏右侧的安装图标
3. 或使用菜单：设置 → 更多信息 → 创建快捷方式

**PWA 功能：**
- 可安装到桌面
- 离线可用（显示最后一张可视化）
- 支持 Windows/macOS/Linux
